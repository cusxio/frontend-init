Frontend Init
Welcome !
Frontend Init is a production ready boilerplate for the perfect developing experience.
It can be easily extended to incorporate the Frontend Framework of your choice.
Completely unopinionated.
Features
...
  • Works with most modern Frontend Frameworks.
  • React · Angular · Inferno · Riot · Vue
  • Or with your favourite JavaScript Templating Engine.
  • Pug · Handlebars · Nunjucks · Hogan · Mustache
  • JavaScript Tree-Shaking with Webpack 2.
  • Hot Reloading goodness.
  • Sass, extendible with PostCSS.
  • Aggressive minification of HTML/CSS/JS.
  • Image optimization and compression with Imagemin.
  • Instant GitHub Pages Deploy.
